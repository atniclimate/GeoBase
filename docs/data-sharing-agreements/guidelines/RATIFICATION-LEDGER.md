# Ratification ledger — RSTEP Tribal Data Sovereignty Guidelines

The Guidelines are DRAFT until this ledger shows completion of the
ratification path for **every Nation attributed in the draft** (not merely
the regional denominator). Each entry pins exact evidence: claim_ids and
content_versions, so a later document version never silently inherits a
prior approval.

## Path (owner-decided 2026-07-21)

1. RSTEP Tribal Advisory Board review → approval recorded here.
2. Direct outreach to each attributed Nation (custodian model per
   `../pnw-data-sovereignty-research.md`: council secretary, legal counsel,
   health/research review office).
3. Tribal IRB review where a Tribal IRB exists.
4. Full ratification.

Corrections requested at any step are handled as catalog `correction`/
`review` events plus draft revisions, and the affected ledger row reverts
to pending.
