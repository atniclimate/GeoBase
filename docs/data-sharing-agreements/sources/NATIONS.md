# Canonical Nation identifiers — coverage denominator

The denominator for this subproject is the 43 federally recognized Tribes
of the WA/OR/ID region served by the Northwest Tribal Epidemiology Center,
per `../pnw-data-sovereignty-research.md` (machine copy:
`nations.json`, version `2026-07-21-nwtec-43`). `nation_id` values in the
register and catalog MUST come from this list. **Owner decision
2026-07-21:** ATNI-roster expansion is deferred to DS-6 / a future
iteration (receipt in `atni-roster-status.json`); when it happens, add rows
to both files with `atni_member` verified and bump the denominator
version.

The coverage matrix (`coverage-matrix.md`, generated at the DS-1 gate) must
account for every row below with exactly one status: `found` |
`searched-not-found` | `inaccessible` | `excluded` | `not-yet-searched`.
"searched-not-found" is never evidence a Nation lacks a policy.
