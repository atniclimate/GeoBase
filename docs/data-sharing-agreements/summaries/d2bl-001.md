## What it is

Northwest Tribal Epidemiology Center Data Governance Handbook, Version 1.1 is cataloged as a `data-governance-policy` issued by Northwest Tribal Epidemiology Center. It covers: Northwest Tribal Epidemiology Center and NPAIHB staff, project directors, data stewards and custodians, contractors, consultants, interns, external collaborators, data requesters, member Tribes, and data providers, but only as each handbook provision specifies.

The cataloged data scope is: Tribal, state, and federal public-health data; record-level PII and PHI; the Northwest Tribal Registry; Tribal Data Hub data; Tribe- and Tribal-area-specific aggregates; analytic files, transfers, reports, publications, media releases, and destruction records.
