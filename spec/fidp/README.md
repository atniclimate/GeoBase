# FIDP — Federated Indigenous Data Protocol (placeholder)

TSDF names a companion technical specification, the **Federated Indigenous Data
Protocol (FIDP)**, for federated data infrastructure. GeoBase's federation layer
(**Phase 2.0** in `docs/ROADMAP.md`) implements against FIDP.

This directory is a placeholder. It will hold GeoBase's FIDP profile: how T0
baselines are advertised, verified, and auto-distributed between federated nodes,
and the architectural guarantees that T1–T3 data never transit the federation.

**Not yet implemented.** No federation code ships before Phase 2.0. Until then,
GeoBase runs as a single grounded node (see `place.example.toml`).
